# 英語の勉強 解説

## 単語
単語は英語力の約半分を占めます。単語がわかればだいたい読めますし、分からなければ読めません。英語学習をするにあたって、単語からは逃げられません。

まずDUO3.0を使います。DUO3.0には単語1600＋熟語1000が含まれていて、この教材を極めることで5000語レベルに到達することができます。5000語というと英検2級、TOEIC600点、センター試験8割レベルです。すべての基本です。せっかくなのでCDを使ってリスニング・ライティング・スピーキング力も上げてしまいます。最終的にはこの本の例文を全部暗記します。暗記することで総合的に英語力を上げます。暗記は辛いですが、英語学習の一番の近道です。

予想所要時間は80時間です。

### DUO3.0 使い方
セクションごとに区切って覚えます。1セクションあたり8～17文を覚えることになります。

１．まず普通に読みます。青い意味を覚えます。黒い派生語はとりあえず目を通すだけでいいです。英文を読んで和訳が出てくるようになるまで繰り返してください。1セクション当たり6-8ページ程度です。

２．少々アレンジした音読パッケージをします。CD音源は1セクションごとに1トラックあります。1トラック当たり1分～1分半程度です。1曲リピートモードにして無限ループしながら２－１～２－３をやりましょう。

２－１．リスニングします。まずは本文を見ながら、意味が分かってきたら本文を外します。本文を見ないでも大体の意味が分かるようになったら２－２に進みます。

２－２．テキストを見ながらオーバーラッピングします。音声に被せるようにして音読してください。文字に対応した正しい発音を学ぶことが目的です。いい感じに言えるようになったら２－３．に進みます。

２－３．テキストを見ないでシャドウイングします。音声に遅れて発音します。機械的に発音するだけではなく、意味を取りながら発音してください。シャドウイングしながらでも意味が取れるようになったら完了です。

３．quizletを使って暗記します。1セクションごとに区切った学習セットを使い、日本文から英語に直します。「オプション」で質問のタイプを単語カードのみにし、英語で応えるようにしてください。正直言って苦しいです。1セクション覚えきったら終わりです。

以上を45セクション全部についてやりましょう。1セクションあたり１～３まで40~50分くらいでできるのではないかと思います。

45セクションやり切ったら、復習しましょう。通読（速くていい）と通し聞き（シャドウイングでもいい）が有効です。quizletでは560文まとめた学習セットを使って復習しましょう。そこまでできたらもう極めたと言っていいでしょう。

### DUO後
DUOはあくまでも単語の入門にすぎません。辞書を引きまくりギリギリ洋書や英字記事が読めるレベルには達しますが、辞書なしでまともに読もうと思ったら単語力は10000-12000語レベルが必要です。単語との戦いは終わることがありません。

次の教材としてはDUOと同様に例文で覚えていく究極の英単語Vol.2（大半は知っているので、穴埋め）→3（ムズイ）→4（超ムズイ）と進むのがよいと思います。もう例文で覚えるのが嫌だと思ったら、文章で覚えていく系の速読英単語上級編→速読速聴英単語Core1900→Advanced1100でもいいです。もちろん、相乗効果は見込めるので、両方使うのもいいです！

## 文法・語法
文法・語法は単語と並ぶ英語力の要です。単語がわかっていても文法ができていなければやはり読めません。語法とは文法の中でも暗記寄りの項目のことですがこれも重要なので覚えましょう。単語、文法・語法がわかれば、あとはこれの応用なので、理論的にはなんでも読めるようになります。

予想所要時間は100時間です。

### 勉強方法
１．ジーニアス総合英語の対応するセクションを読みます。解説を熟読して理解しましょう。そしてここで一番重要なのですが、基本例文862文は、すべて覚えてもらいます。本当は基本文以外の例文も全部覚えたほうがいいのですが、多すぎていやになるのでカットします。音声も活用しましょう。DUOと同じ方法で、リスニング・オーバーラッピング・シャドウイングするのがよいです。

２．ここが一番苦しいところですが、DUOと同様にquizletでセクションごとの例文を覚えます。なぜ覚えるかというと、ここで覚えると後の問題演習が超楽になるからです。

３．Vintageの対応する問題を解きます。１．２．で頑張れば楽勝で解けるはずです。解説はしっかり読んで、よくわからなければジーニアス総合英語に戻りましょう。基本例文ではない例文に対応している問題もあるので、その都度覚えましょう。セクションごとの問題数はあまり多くないので、正答率が悪ければ2周してもよいです。

４．完全を期すため、頻出英文法・語法問題1000の対応する問題も解きましょう。解説は詳しいので、ばっちり読んでおきましょう。

５．１～４を１単元ごと（時制・完了形とか、助動詞とか）に順番に行います。一度にジーニアス総合英語を読むのは大変なので、少しずつ進みましょう。

### 注意点
ジーニアス総合英語の1,2はほぼ中学生の復習なので、DUOに入る前にここでリスニング・オーバーラッピング・シャドウイングおよび例文暗記の練習をやることをお勧めします。はじめてみるとおそらく、びっくりするくらい聴き取れないし、暗記できません。そういうもんです。

また、ジーニアス総合英語には動詞の語法については一切解説がありません。こればかりは問題を解いて覚えるしかないです。ジーニアス総合英語は動詞以外については語法の解説がかなり充実している方です。

## リスニング
「みるみる英語力がアップする音読パッケージトレーニング」を使います。まずは説明部分を通読します。非常にわかりにくい説明です。特に裏表紙、セクションごとなのか通しでやるのかどっちなんだよ！不安になりますが頑張って解釈し、テキスト通りにトレーニングしましょう。

予想所要時間は100時間です。

１．テキストを一切見ないで通し聞きをします。CDにはゆっくり音声とナチュラル音声が入っていますが、あまり速度に変わりがないので、ナチュラル音声でいいでしょう。41分→35分と6分短縮になります。何日かかけて何回も聴いて、もうこれ以上分からないというところまで続けるそうです。なおテキストには「中学レベル」と書いてありますがざっと見たところ高1修了レベルだと思いました。全く歯が立たない、無理と思ったならこの教材はDUOが終わってからにしましょう。

２．メインパートに入ります。ここからは「サイクル」という単位で教材を回していくそうです。

２－１．聴き解きをします。1センテンス・1フレーズごとに文と音を照合して答え合わせをします。1サイクル目のみ。

２－２．テキストを見ないでリピーティングをします。どれだけできないか絶望するためにやるのではなく、２－５でどれくらいできるようになったかわかるようにするためです。1回。

２－３．テキストを見ながらリピーティングをします。3～5回。

２－４．音読します。1サイクル目は個人差がありますが10～15回。

２－５．テキストを見ないでリピーティングをします。3回程度。

２－６．ナチュラルスピード音源でシャドーイングをします。3～5回。

３．１サイクル目は２－１～２－６をセクションごとに行い、２サイクル目以降は２－２～２－６を行います。ということを理解するまで時間がかかりました。1サイクル目は1つのセクションを30回程度繰り返すことになるので、1セクション当たり1時間程度かかります。40セクションあるので40時間。2サイクル目からは回数を減らしていいとはいえ、初学者はこれを7サイクルこなせと書いてあります。概算で150時間程度かかります。正気とは思えませんが英語学習とはそういうものらしいです。さすがにもう少し少なくしてもいいと思います。この本には2サイクル目以降はテキスト全体で回せと書いてありますが、全体で回すのはとても時間がかかる（特に、リピーティングは1回80分かかる）ので、まず無理です。普通にセクションごとでよいでしょう。記憶してしまうからダメって言ってるけど、テキスト全体で回すとか意味わかんない。web上にある元ネタの「英語完全上達マップ」にもテキスト全体で回せとは書いてないので、セクションごとでいいと思います。

## 全体の学習配分

まず文法の「1 文の種類」「2 動詞と文型」をウォームアップでやり、その後は単語と文法を併用して進めていくのがよいと思います。単語、文法とも1時間程度を一つの区切りにして1日2時間目標で進めていくとわかりやすいかも。リスニングも1時間単位で並行してやってもいいですが、辛かったら単語・文法のどちらかが終わってからの後回しでいいでしょう。単語・文法とも音読パッケージを取り入れているので、リスニング対策で30回も繰り返さなくていいレベルになるのでは、と思います。
